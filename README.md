# Digital Library System

## โครงการนี้คืออะไร

ระบบห้องสมุดดิจิทัล (Digital Library System) เป็นเว็บแอปพลิเคชันที่พัฒนาด้วย Django สำหรับการจัดการห้องสมุดออนไลน์ ระบบนี้ช่วยให้ผู้ใช้สามารถยืม-คืนหนังสือ จองหนังสือ และจัดการค่าปรับได้อย่างมีประสิทธิภาพ

### ฟีเจอร์หลัก

- **การจัดการหนังสือ (Books)**: ค้นหา เรียกดู และดูรายละเอียดหนังสือ
- **ระบบยืม-คืน (Loans)**: ยืมหนังสือและติดตามสถานะการยืม
- **ระบบจอง (Reservations)**: จองหนังสือที่ไม่ว่างและจัดการคิวการจอง
- **ตะกร้าหนังสือ (Shopping Cart)**: เพิ่มหนังสือหลายเล่มก่อนยืมพร้อมกัน
- **ค่าปรับ (Fines)**: ติดตามและจัดการค่าปรับกรณีคืนหนังสือล่าช้า
- **ระบบผู้ใช้ (Users)**: ลงทะเบียน เข้าสู่ระบบ และจัดการบัญชีผู้ใช้
- **แดชบอร์ดผู้ดูแล**: จัดการหนังสือ ผู้ใช้ และการยืม-คืน

## วิธีการ Run Project

### ข้อกำหนดเบื้องต้น

- Python 3.8 ขึ้นไป
- pip (Python package manager)
- Virtual environment (แนะนำ)

### ขั้นตอนการติดตั้งและรัน

#### 1. Clone โครงการ

```bash
git clone <repository-url>
cd digital-library
```

#### 2. สร้าง Virtual Environment

```bash
python -m venv .venv
```

#### 3. เปิดใช้งาน Virtual Environment

**macOS/Linux:**
```bash
source .venv/bin/activate
```

**Windows:**
```bash
.venv\Scripts\activate
```

#### 4. ติดตั้ง Dependencies

```bash
pip install -r requirements.txt
```

#### 5. ตั้งค่าฐานข้อมูล

```bash
python manage.py migrate
```

#### 6. สร้างผู้ใช้งาน Admin (ถ้าต้องการ)

```bash
python manage.py createsuperuser
```

ตอบคำถามตามที่โปรแกรมถาม (username, email, password)

#### 7. รันเซิร์ฟเวอร์

```bash
python manage.py runserver
```

#### 8. เข้าใช้งานระบบ

เปิดเบราว์เซอร์และเข้าไปที่:

- **หน้าหลัก**: http://127.0.0.1:8000/
- **หน้า Admin**: http://127.0.0.1:8000/admin/

## โครงสร้างโครงการ

- `books/`: แอปพลิเคชันจัดการหนังสือและตะกร้า
- `loans/`: แอปพลิเคชันจัดการการยืม-คืน
- `reservations/`: แอปพลิเคชันจัดการการจองหนังสือ
- `fines/`: แอปพลิเคชันจัดการค่าปรับ
- `users/`: แอปพลิเคชันจัดการผู้ใช้และ authentication
- `config/`: การตั้งค่า Django หลัก
- `templates/`: ไฟล์ HTML templates
- `media/`: ไฟล์รูปภาพปก ของหนังสือ
- `docs/`: เอกสารประกอบโครงการ

## การใช้งานพื้นฐาน

### สำหรับผู้ใช้ทั่วไป

1. ลงทะเบียนบัญชีผู้ใช้ใหม่
2. เข้าสู่ระบบ
3. เรียกดูและค้นหาหนังสือ
4. เพิ่มหนังสือลงในตะกร้า
5. ยืมหนังสือจากตะกร้า
6. ตรวจสอบสถานะการยืมและค่าปรับของคุณ

### สำหรับผู้ดูแลระบบ

1. เข้าสู่ระบบด้วยบัญชี superuser
2. เข้าไปที่ /admin/ เพื่อจัดการ:
   - หนังสือและหมวดหมู่
   - ผู้ใช้และสิทธิ์การใช้งาน
   - การยืม-คืนและการจอง
   - ค่าปรับและการชำระเงิน

## เทคโนโลยีที่ใช้

- **Django 4.2**: Web framework หลัก
- **SQLite**: ฐานข้อมูล
- **Pillow**: จัดการรูปภาพปกหนังสือ
- **HTML/CSS**: Frontend templates

## หมายเหตุ

- ข้อมูลจะถูกเก็บในไฟล์ `db.sqlite3`
- รูปภาพปกหนังสือจะถูกเก็บในโฟลเดอร์ `media/books/covers/`
- เอกสารเพิ่มเติมสามารถดูได้ในโฟลเดอร์ `docs/`
